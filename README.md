# 🏋️ Apex Fitness - Premium Landing Page

A high-performance, visually stunning landing page designed for a premium fitness coaching brand. This project focuses on delivering an elite user experience through smooth scrolling, 3D magnetic interactions, and dynamic reveal animations, all while maintaining a flawless dark aesthetic.

## 🌟 Key Features

- **Premium Dark Aesthetic:** A sleek, modern design with custom glowing borders, cinematic gradients, and a bespoke noise overlay (`grain`) to create a premium feel.
- **Magnetic UI Elements:** Custom React hooks (`useMagneticButton`) that create an interactive, physical feel by pulling buttons and call-to-action elements toward the user's cursor.
- **Smooth Scrolling:** Hand-crafted smooth scroll logic that perfectly frames content sections while intelligently managing large vertical padding for optimal viewing on any screen size.
- **Dynamic Scroll Animations:** Custom Intersection Observers (`useScrollReveal`) that orchestrate beautiful entrance animations as elements come into the viewport.
- **3D Tilt Cards:** Interactive service and pricing cards that physically tilt based on mouse movement (`useTiltEffect`), adding depth and engagement to the design.
- **Live Counters:** Animated numeric counters that build trust and showcase social proof as the user scrolls down.
- **High-Performance Architecture:** Built on Vite, React 18, and TypeScript, guaranteeing lightning-fast HMR during development and heavily optimized bundles for production.

## 🛠️ Tech Stack

- **Core:** React 18, TypeScript, Vite
- **Styling:** Custom CSS with CSS Variables for theme consistency, Flexbox/Grid for responsive layouts.
- **Icons:** Lucide React
- **Animations:** Native Web Animations API, CSS Transitions, Custom JS Hooks

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository** (if you haven't already).
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Open your browser** and navigate to `http://localhost:5173`.

## 📦 Deployment

This project is optimized and ready to be deployed to platforms like Vercel, Netlify, or Cloudflare Pages.

1. **Build the project:**
   ```bash
   npm run build
   ```
2. **Deploy the `dist` directory** generated by the build process.
